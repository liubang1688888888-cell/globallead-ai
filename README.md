# GlobalLead AI V1.0

第一版演示版本。GlobalLead AI 是一个用于海外客户开发演示的 AI 获客平台，支持搜索真实企业线索、AI 推荐排序、CRM 管理、AI 开发信生成，并一键打开常用外联渠道。

## 版本

GlobalLead AI V1.0

## 主要功能

- AI客户搜索
- AI推荐
- CRM
- AI开发信
- Gmail
- Outlook
- WhatsApp
- LinkedIn
- Excel
- CSV

## 说明

这是第一版演示版本。V1 代码已冻结，后续开发基于 V2 分支进行。

## 启动

```bash
npm install
npm start
```

默认地址：

```bash
http://127.0.0.1:3000
```
